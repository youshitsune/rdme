# RDME

## Description

A C program that prints formatted README.md.

## Installation

Clone it.
```bash
git clone https://github.com/youshitsune/rdme
```

And make it.
```bash
cd rdme/
sudo make install
```

## Usage

Run:
```bash
rdme -h
```
